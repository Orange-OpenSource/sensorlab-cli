# sensorlab-collector

SensorLab2 collector


## Brief

This node module collects logs on a MQTT server and consolidates it
 either in a JSON or PCAP file.

## Installation

sensorlab-commands is available via NPM:
 
```bash
    npm install sensorlab-collector --save
```
